Proyecto de BigData

Para la Maestría de Analítica de Datos

¿Qué es nuestro proyecto?

Este proyecto de BigData está diseñado para ofrecer soluciones integrales en la gestión y análisis de grandes volúmenes de datos. Gracias a la combinación de tecnologías como MongoDB para almacenamiento flexible, Elasticsearch para búsquedas potentes y Flask para un backend ligero, hemos construido una plataforma donde la creación de bases de datos, la indexación de documentos y la visualización de resultados son procesos sencillos e intuitivos.

Nuestro objetivo es facilitar a estudiantes y profesionales del área de Data Analytics una interfaz amigable para explorar, consultar y gestionar datos de manera eficiente, con un enfoque en rendimiento y escalabilidad.

Características Destacadas

Gestión de Bases de Datos

Crea y administra múltiples bases de datos y colecciones de forma sencilla usando un entorno web. Consulta el conteo de documentos, elimina y crea colecciones con un solo clic.

Indexación en Elasticsearch

Sube archivos JSON comprimidos en formato ZIP y automatiza la indexación en Elasticsearch. Aprovecha potentes búsquedas de texto completo y filtros avanzados.

Búsqueda Avanzada

Realiza consultas de búsqueda por texto o por campo específico, además de filtros por rango de fechas. Visualiza tendencias mediante histogramas y agrupaciones por categorías.
